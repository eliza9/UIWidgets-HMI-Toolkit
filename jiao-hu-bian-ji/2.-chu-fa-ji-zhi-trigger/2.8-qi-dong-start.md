# 2.8 启动 Start

### ![](../../.gitbook/assets/Start.png)

### 功能描述：

触发交互动作：启动直接触发。

该功能即用来配置进入当前Screen的状态。



### 设置属性：

可以输入信号内容配置



### 可支持Target：

按场景支持模拟信号，支持 Screen、Test、3D 模型。
