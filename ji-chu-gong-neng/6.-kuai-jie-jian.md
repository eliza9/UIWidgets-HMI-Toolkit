# 6. 快捷键

同其他设计软件，本编辑器设置了符合设计师使用习惯的快捷键

| 快捷键       | 功能说明                                                        |
| --------- | ----------------------------------------------------------- |
| Ctrl+C    | <p>选中Screen时，拷贝当前Screen；<br>选中元件图层，当前元件图层</p>               |
| Ctrl+V    | <p>鼠标区域在视图范围内时，粘贴Screen在视图区域内</p><p>鼠标区域在画板中，粘贴元件图层在画板中</p> |
| Ctrl+Z    | 撤回上一步操作                                                     |
| Ctrl+鼠标点击 | 多选，支持再次右键，有快捷操作                                             |

（暂时：苹果电脑也只支持Ctrl键）
